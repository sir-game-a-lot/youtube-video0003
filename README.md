## Prerequisites

A fresh install of Proxmox - you can follow this tutorial on older installs too.

Make sure it is accessible from the web browser.

2x Spinning Hard-drives, 2 TB or higher (per drive) recommended.

## Checklist

1. System Updates and Initial Configuration
2. Harden Security of the system
3. Storage Planning and Configuration
4. Backups!
